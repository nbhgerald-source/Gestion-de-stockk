Importer le stock initial (CSV)

Chargez un fichier CSV (delimiteur ;, encodage UTF-8) pour enregistrer les quantites de depart de chaque produit, depot par depot. Chaque ligne cree une entree de stock (un lot), comme une saisie manuelle d'entree. Telechargez d'abord le modele :

Telecharger le modele CSV

  • code_produit : code d'un produit existant (creez/importez vos produits avant ceci).
  • depot_code : code d'un depot existant (Parametres > Depots).
  • quantite / cout_unitaire : exprimes dans l'unite de GESTION du produit.
  • date_entree : optionnelle (AAAA-MM-JJ), aujourd'hui par defaut.
  • numero_lot / reference : optionnels.
  • En cas d'erreur sur une seule ligne, aucune entree n'est enregistree (import tout-ou-rien).

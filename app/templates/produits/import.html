Importer des produits (CSV)

Chargez un fichier CSV (delimiteur ;, encodage UTF-8) contenant l'ensemble des articles a creer. Telechargez d'abord le modele pour respecter le format attendu :

Telecharger le modele CSV

  • type_produit : MATIERE_PREMIERE, ACCESSOIRE, SEMI_FINI, RESIDUEL ou PRODUIT_FINI.
  • unite_achat_code / unite_gestion_code : codes d'unites existantes (Parametres > Unites de mesure).
  • facteur_conversion : nombre d'unites de gestion par unite d'achat (1 si identiques).
  • Un produit dont le code existe deja sera rejete (aucun doublon).
  • En cas d'erreur sur une seule ligne, aucun produit n'est cree (import tout-ou-rien).
